# TypeScript Web Scraper

A fully typed web scraper for energy drink deals from mydealz.de using TypeScript, Playwright, and proper type safety.

## Features

- **Full TypeScript Support**: Strict type checking with comprehensive interfaces
- **Type-Safe Configuration**: Strongly typed configuration options
- **Robust Error Handling**: Custom error types and proper error propagation
- **Modular Design**: Separated types, utilities, and main scraper logic
- **Export Support**: Can be used as a module or run directly
